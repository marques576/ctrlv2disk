# 📋 Clipboard Image Saver

A simple Python utility that grabs an image from your clipboard and saves it as a PNG file with either a custom name or an automatic timestamp.

---

## ✨ Features

- 📸 Captures images directly from your clipboard  
- 📝 Lets you choose a custom filename  
- ⏱️ Automatically generates a timestamp if no name is provided  
- 💾 Saves images in PNG format  
- ⚡ Lightweight and easy to use
